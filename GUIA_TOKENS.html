📖 Guia Completo: Gerar Tokens

Passo a passo para gerar seus tokens do GitHub e Vercel

🐙 Como Gerar Token GitHub

1 Abra GitHub
Acesse github.com e faça login com sua conta
2 Clique na sua foto de perfil
Canto superior direito da página → Clique na sua foto/avatar
3 Acesse "Settings"
No menu que aparecer, clique em "Settings" (Configurações)
4 Vá para "Developer settings"
No menu esquerdo, desça até o final e clique em "Developer settings"
5 Acesse "Personal access tokens"
Clique em "Personal access tokens" → "Tokens (classic)"
6 Clique "Generate new token"
Botão azul "Generate new token" → "Generate new token (classic)"
7 Preencha as informações
Token name:
d10-barbearia-deploy
Expiration: Selecione "90 days" (ou mais)
8 Selecione as permissões
Marque os checkboxes abaixo:
  • repo (acesso completo aos repositórios)
  • workflow (GitHub Actions)
  • user (dados do usuário)
9 Clique "Generate token"
Botão verde no final da página
10 Copie o token
Aparecerá um código longo começando com ghp_
⚠️ COPIE IMEDIATAMENTE! Ele não aparecerá novamente!
✅ Pronto! Você tem seu token GitHub. Cole-o no arquivo CREDENCIAIS_SEGURAS.html

⚡ Como Gerar Token Vercel

1 Abra Vercel
Acesse vercel.com e faça login (pode usar GitHub login)
2 Clique na sua foto/iniciais
Canto inferior esquerdo (se em desktop) ou canto superior direito (mobile)
3 Selecione "Settings"
Clique em "Settings" no menu que aparecer
4 Vá para "Tokens"
No menu esquerdo, procure e clique em "Tokens"
5 Clique "Create Token"
Botão "Create" ou "New Token"
6 Preencha o nome
Token name:
d10-barbearia-deploy
7 Defina expiração
Expiration: Selecione "90 days" (ou deixe sem expiração)
8 Clique "Create"
Botão para gerar o token
9 Copie o token
Aparecerá um código começando com vercel_
⚠️ COPIE IMEDIATAMENTE! Ele não aparecerá novamente!
✅ Pronto! Você tem seu token Vercel. Cole-o no arquivo CREDENCIAIS_SEGURAS.html

✅ Próximos Passos

📋 Você já tem os dois tokens? Ótimo! Agora faça o seguinte:
1 Abra CREDENCIAIS_SEGURAS.html
Arquivo que criamos anteriormente na pasta do projeto
2 Cole seus tokens
  • GitHub Personal Access Token (começa com ghp_)
  • Vercel API Token (começa com vercel_)
  • Seu email do GitHub
  • Nome do repositório (d10-barbearia-app)
3 Crie uma senha forte
Exemplo:
D10@2026#Seguro!
Mínimo 6 caracteres (use maiúsculas, números, símbolos)
4 Clique "🔒 Criptografar & Salvar"
Seus dados serão criptografados no navegador
5 Copie o código criptografado
Botão "📋 Copiar para Clipboard"
6 Compartilhe comigo
Envie por CANAIS SEPARADOS:
• Código criptografado via Slack/email
• Senha por WhatsApp/chat pessoal
