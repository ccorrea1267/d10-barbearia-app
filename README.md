## 🚀 Como fazer deploy na Vercel

### Passo 1: Criar repositório no GitHub

1. Acesse [github.com](https://github.com) e faça login.
2. Clique em "New repository".
3. Nome: `d10-salao-app` ou mantenha o nome antigo se preferir.
4. Descrição: "App premium de agendamento para Salão de Beleza".
5. Selecione "Public" ou "Private".
6. Clique em "Create repository".

### Passo 2: Fazer push do código

Abra o terminal na pasta `08-D10SalaoBeleza` e execute:

```bash
git remote add origin https://github.com/ccorrea1267/d10-salao-app.git
git branch -M main
git push -u origin main
```

### Passo 3: Conectar à Vercel

1. Acesse [vercel.com](https://vercel.com).
2. Faça login com GitHub.
3. Clique "Add New..." → "Project".
4. Selecione seu repositório.
5. Clique em "Import".
6. O deploy será feito automaticamente!
