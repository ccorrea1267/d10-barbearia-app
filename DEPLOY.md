# 🚀 Guia Rápido: Deploy na Vercel

## ⚡ 3 Passos Simples

### 1️⃣ GitHub - Criar Repositório

```
1. Va para github.com
2. Clique em "New repository"
3. Nome: d10-barbearia-app
4. Clique "Create repository"
5. Copie os comandos que aparecem
```

### 2️⃣ Git - Enviar Código

Abra **PowerShell** ou **Terminal** na pasta `d10-barbearia-app` e rode:

```bash
git remote add origin https://github.com/SEU_USUARIO/d10-barbearia-app.git
git branch -M main
git push -u origin main
```

⚠️ Troque `SEU_USUARIO` pelo seu username do GitHub!

### 3️⃣ Vercel - Deploy Automático

```
1. Va para vercel.com
2. Clique "Add New..." → "Project"
3. Selecione seu repositório
4. Clique "Import"
5. Aguarde ~1 min
6. Copie o link (tipo: https://d10-barbearia-app.vercel.app)
```

## ✅ Pronto!

Seu cliente pode ver o app em tempo real no link da Vercel!
